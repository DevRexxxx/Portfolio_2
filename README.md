## ✨ Overview

This repository houses the personal portfolio website of **Eshaan Singh Deo**. Designed with a focus on modern web aesthetics (glassmorphism, micro-animations, and dynamic 3D elements), it serves as a central hub for exploring my professional background, technical skills, and featured software projects.

## 🏗️ Architecture & Tech Stack

The portfolio is built on the cutting-edge **Next.js App Router** with a clean, modular component architecture:

### Tech Stack

| Layer | Technology |
|---|---|
| **Framework** | Next.js 16 (App Router) |
| **UI Library** | React 19 + TypeScript 5 |
| **Styling** | CSS Modules + Global CSS variables |
| **Animations** | Framer Motion (Scroll reveals & micro-interactions) |
| **Data Fetching** | GitHub REST API (Resume/Repos integration) |

## 🔧 Installation & Development

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Git](https://git-scm.com/)

### Setup

```bash
# Clone the repository
git clone https://github.com/DevRexxxx/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start dev server with hot reload
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
